# 🎂 Age Calculator (Python)

## 📌 Description
This is a simple yet powerful Age Calculator built using Python.  
It takes the user's birth date as input and calculates the **exact age in years, months, and days** using the current system date.

---

## 🚀 Features
- 🧾 User input (name and birth date)
- 📅 Real-time age calculation
- ⏳ Exact age in years, months, and days
- ⚠️ Handles invalid input
- 🧠 Uses Python's datetime module

---

## 🛠️ Technologies Used
- Python (Beginner Friendly)
- datetime module
- Input/Output functions
- Conditional statements
